About App

The reviews helps the user to make better decisions regarding their purchase, and if you can get the affirmation on that reviews if that would be useful to others or not, that would be like cherry on the cake.

This app helps in validating if the review provided by user will be helpful to others or not. From the past reviews, we have build machine learning model which verify the reviews and encourage the users to provide the helpful reviews.
